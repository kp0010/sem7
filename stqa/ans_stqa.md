### McCalls's Quality Factors
Revision:
- Maintainibility
- Testability
- Flexibility

Operation:
- Correctness
- Usability
- Reliability
- Efficiency
- Integrity

Transition:
- Portability
- Interoperability
- Reusability

### Type of Metrics
1. Product Metrics
- Lines of Code (LOC)
- Cyclomatic Complexity
- Function Points
- Code Coverage
- Defect Density
- Maintainability Index

2. Process Metrics
- Defect arrival rate
- Review efficiency
- Mean Time to Repair (MTTR)
- Schedule Variance
- Process cycle time
- Cost of Quality

3. Project Metrics
- Team size
- Planned vs Actual cost
- Resource utilization
- Effort (person-hours/person-months)
- Risk metrics
- Delivery milestone tracking

4. Size Metrics
- Lines of Code (LOC)
- Number of Classes or Modules
- Function Point Analysis (FPA)
- Use Case Points

5. Complexity Metrics
- Cyclomatic Complexity
- Cognitive Complexity
- Halstead Complexity Metrics

6. Quality Metrics
- Defect Density
- Mean Time Between Failures (MTBF)
- Customer-reported defects
- Reliability Index
- Code readability score

7. Internal Metrics
- Coupling and Cohesion
- Code complexity
- Comment density
- Fan-in/Fan-out values

8. External Metrics
- Response time
- System availability
- Operational failure rate
- User satisfaction score
